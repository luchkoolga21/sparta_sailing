## 🌟 12. ВИЗУАЛЬНАЯ АЙДЕНТИКА

### Шрифты
- **Заголовки:** Inter (Bold, 700) с плотным letter-spacing (-3.2px)
- **Текст:** Inter (Regular, 400) с легким letter-spacing (-0.32px)
- **Акценты:** Roboto Mono (Medium, 500) для кнопок и меток

### Иконки
- Стиль: Outline (контурные, тонкие линии)
- Библиотека: Solar Icons (Iconify)
- Цвет: #2C7A7B (бирюзовый) для акцентов

### Border Radius
- Кнопки: 6px (маленький, четкий)
- Карточки/Фото: 24-40px (большой, мягкий, премиальный)

### Spacing
- Между секциями: 120-200px (очень щедрый, как brand.ai)
- Grid gap: 30px
- Контейнер: max-width 1040px, padding 80px
