## 🎯 Стилистические требования к фото

- **Цветовая гамма Day Mode:** Яркие, насыщенные, бирюзовые/синие тона
- **Цветовая гамма Sunset Mode:** Теплые, приглушенные, оранжево-фиолетовые тона
- **Эстетика:** Премиальность, но без вычурности. Атмосферность > гламур
- **POV-фото:** Должны создавать ощущение "Я уже там"
- **Формат:** JPG для фото, SVG для логотипа, иконки через Iconify CDN

---

## 📝 Примечания

- Все фото должны быть оптимизированы для веба (сжатие без потери качества)
- Иконки загружаются через Iconify CDN (не нужно скачивать отдельно)
- Для временной разработки можно использовать placeholder-сервисы (например, `https://placehold.co/`)
- Финальные фото будут заменены реальными снимками из Находки/Приморья
